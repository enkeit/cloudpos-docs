# Getting Started

## Logging In

Navigate to your admin URL and enter your email and password. The default admin account is created during initial setup by your IT team.

## Overview of the Interface

After login you see:
- **Top navigation** — Quick links, notifications, user profile
- **Sidebar** — Main navigation (Products, Customers, Reports, Settings, etc.)
- **Main content area** — Active page

## First-Time Setup Checklist

1. **Company Settings** — Add your business name, logo, tax number, and address under *Settings → Company*
2. **Store Settings** — Configure your store name and branch details
3. **Payment Methods** — Enable Cash, Card, UPI, or custom methods under *Settings → Payment Methods*
4. **Tax Rules** — Set up VAT/GST rates under *Settings → Tax*
5. **Add Products** — Import or manually add your product catalog
6. **Add Staff** — Create user accounts for cashiers and managers
7. **Connect POS App** — Share the API key with staff to link the mobile/desktop app
