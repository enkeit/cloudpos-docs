# Accessing Master Data

Navigate to:

**Settings → Master Data**

From here, you can configure various master records used throughout the system.

---

# Available Master Data

## 1. Product Units

Defines the measurement units used for products.

**Purpose**

* Standardize product quantities
* Support inventory and sales calculations
* Enable multiple selling units

### Examples

| Unit   | Description |
| ------ | ----------- |
| PCS    | Pieces      |
| KG     | Kilogram    |
| Gram   | Gram        |
| Ltr    | Litre       |
| ml     | Millilitre  |
| Dozen  | 12 Units    |
| Pack   | Pack        |
| Box    | Box         |
| Carton | Carton      |
| Bottle | Bottle      |

**Used In**

* Product Master
* Purchase
* Sales
* Inventory

# 3. Payment Types

| --------------- | -------------------------- |

**Used In**

* Sales

* Customer Accounts
