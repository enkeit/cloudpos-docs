# Purchases

## Recording a Purchase

1. Go to **Purchases → Add Purchase**.
2. Select the **Supplier**.
3. Add products and quantities received.
4. Enter the purchase price per unit.
5. Set the invoice date and reference number.
6. Save — stock is automatically updated.

## Suppliers

Manage your suppliers under **Purchases → Suppliers**. Each supplier has:
- Name, contact, and address
- Outstanding balance (if on credit terms)
- Purchase history

## Purchase Returns

If you need to return items to a supplier:
1. Go to the original purchase record.
2. Tap **Return** and select items and quantities.
3. Save — stock is decremented and a debit note is created.

## Purchase Vouchers

Purchase vouchers record payments made to suppliers. Create them under **Purchases → Vouchers** to track outstanding payables.
