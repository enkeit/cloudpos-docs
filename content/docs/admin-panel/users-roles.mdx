# Users & Roles

## User Roles

| Role | Access Level |
|------|-------------|
| Super Admin | Full access to all stores and settings |
| Company Admin | Full access to own company |
| Store Manager | Manage a single store; cannot change company settings |
| Sales Executive | Can view own sales data |
| Cashier | POS app only; limited admin access |

## Adding a User

1. Go to **Users → Add User**.
2. Enter name, email, and assign a role.
3. Assign the user to one or more stores.
4. Save — the user receives an email invite to set their password.

## Editing Permissions

Role-level permissions can be customised under **Settings → Roles**. Toggle on/off specific modules (Reports, Customers, Purchases, etc.) for each role.

## Deactivating a User

To revoke access without deleting the account, toggle the **Active** switch on the user's profile. Inactive users cannot log in.
