# Settings

## Company Settings

- **Business name**, logo, and contact details
- **Tax registration number** (VAT/GST)
- Printed on all receipts and invoices

## Store Settings

Each physical store/branch can have its own:
- Store name and address
- Operating hours
- Currency and locale
- Kiosk mode toggle

## Payment Methods

Enable or disable payment methods that appear on the POS checkout:
- Cash
- Card (with or without Pine Labs integration)
- UPI / QR
- Custom methods (Bank Transfer, Credit, etc.)

## Tax Configuration

Create tax rules under **Settings → Tax**:
- Tax name (e.g. VAT 5%)
- Tax rate (%)
- Apply to all products or specific categories

## Receipt & Document Settings

- Upload store logo for receipts
- Select default receipt layout
- Configure footer text (return policy, social handles, etc.)
- Set invoice numbering prefix and starting number

## Discount Rules

Create predefined discount rules under **Settings → Discounts** to allow cashiers to apply approved discounts without manual entry.

## Notification Settings

Configure push notification triggers for events like low stock, new orders, and daily close reminders.
