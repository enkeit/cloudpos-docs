# Dashboard

The Dashboard gives a real-time snapshot of business performance.

## What's on the Dashboard

- **Today's Sales** — Total revenue for the current day
- **Orders Count** — Number of invoices processed
- **Top Products** — Best-selling items by quantity or revenue
- **Sales Chart** — Line chart showing sales trend over the past 7 or 30 days
- **Low Stock Alerts** — Products that need restocking

## Role-Based Views

| Role | Dashboard View |
|------|---------------|
| Admin | Full company overview |
| Store Manager | Store-level metrics |
| Sales Executive | Personal sales summary |
| Cashier | Today's session summary |

## Refreshing Data

Pull down on the dashboard to refresh. Data syncs automatically every few minutes when connected to the internet.
