# Getting Started

## 1. Installation

Download the Enke POS app from the Google Play Store, Apple App Store, or install the Windows desktop build from your administrator.

## 2. First Launch — API Key

On first launch you will be asked for an **API Key** or **Base URL**. This is provided by your system administrator and points the app to your business's server.

## 3. Login

Enter your username and password. If you have forgotten your password, use the **Forgot Password** link to reset it via email.

## 4. Select a Store

If your account has access to multiple stores, you will be prompted to select which store to open a session for.

## 5. Home Screen

After login you land on the **Home Screen** where you can:

- Browse and search products by category
- Open the billing/cart screen
- View pending orders
- Access the sidebar for settings and reports

## Keyboard Shortcuts (Desktop)

The Windows desktop version supports keyboard shortcuts for fast cashier workflows. Press **?** on the billing page to view available shortcuts.
