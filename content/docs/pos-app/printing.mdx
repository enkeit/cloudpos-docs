# Printing

Enke POS supports both **Bluetooth thermal printers** and **standard network/USB printers**.

## Connecting a Printer

1. Go to **Settings → Printers**.
2. For Bluetooth printers, tap **Scan** to discover nearby devices and pair.
3. For network printers, enter the IP address and port.

## Receipt Layouts

Multiple receipt layouts are available:

| Layout | Description |
|--------|-------------|
| Classic | Simple, minimal receipt |
| Standard | Full receipt with logo |
| Premium | Rich design with itemised tax |
| Bilingual | English + Arabic |
| Arabic + English 3 | Compact bilingual |
| Multi-Store | For chains with multiple outlets |

Select a layout in **Settings → Receipt Layout**.

## KOT Printing (Restaurant)

Kitchen Order Tickets (KOT) are sent to the kitchen printer when a table order is confirmed. The KOT shows item names, quantities, and table number.

## Barcode Printing

Go to **Settings → Barcode Printer** to configure barcode label printing. Supports custom label sizes and layouts.

## WhatsApp Receipt

To send a receipt via WhatsApp, tap the **WhatsApp** button on the payment success screen. The customer's number is auto-filled if they are registered.
