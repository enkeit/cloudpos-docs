# POS App Overview

The **Enke POS App** is a Flutter-based point-of-sale application that runs on Android, iOS, and Windows. It is designed for retail and restaurant businesses.

## Key Features

- **Billing & Checkout** — Fast order entry, cart management, and payment processing
- **Customer Management** — Customer profiles, loyalty points, purchase history
- **Product Catalog** — Categories, variants, stock tracking
- **Kitchen Display (KOT)** — Live kitchen order tickets for restaurants
- **Kiosk Mode** — Self-service ordering for customers
- **Reports** — Sales, stock, and supplier reports
- **Receipt Printing** — Thermal and standard printer support with multiple layouts
- **Offline Support** — Core billing works without internet
